## Usage in T2 Funnels

In the funnel-chat system prompt or generated copy, use the `author_image_url` config field:
```json
{
  "config": {
    "author_name": "Dr. Sarah Mitchell",
    "author_credential": "Board-Certified Dermatologist",
    "author_image_url": "https://raw.githubusercontent.com/contextarchitect/brand-assets/main/thumbnails/1776096891709-6p9j1cxfbn5.png"
  }
}
```
