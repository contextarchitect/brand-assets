## Usage in T2 Funnels

In the funnel-chat system prompt or generated copy, use the `author_image_url` config field:
```json
{
  "config": {
    "author_image_url": "https://raw.githubusercontent.com/contextarchitect/brand-assets/main/thumbnails/f1-middle-eastern-medical.png"
  }
}
```
