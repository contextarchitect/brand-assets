# PROTOCEL — Product Reference Images

**Brand:** PROTOCEL
**Purpose:** Publicly accessible product photography for AI image generation APIs (Kie.ai Nano Banana Pro, Creative Engine).

**Important:** This repository is PUBLIC. Only product images that are already visible on the brand's website or ads should be stored here. Never upload strategy documents, internal files, or unreleased product imagery.

**Usage:** Reference these images by their raw URL:
```
https://raw.githubusercontent.com/contextarchitect/brand-assets/main/protocel/product-images/[filename]
```

**Full index:** See `protocel/image-reference-index.md` in the private `context-architect-brands` repo for the complete catalog with REF numbers, categories, scale context assessments, and avatar matching.
