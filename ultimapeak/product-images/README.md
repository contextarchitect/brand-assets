# UltimaPeak Product Images

This folder contains product photography and logo files for UltimaPeak Performance Gummies. These images are publicly accessible for use by AI image generation APIs.

## Contents

- Product photos (7 files) — pack and gummy shots with scale context
- Logo variants (3 files) — black, white, and negative versions
- Studio product shots (2 files) — isolated front/back packaging

## Usage

Public URL pattern:
```
https://raw.githubusercontent.com/contextarchitect/brand-assets/main/ultimapeak/product-images/[filename]
```

See the full Image Reference Index in the UltimaPeak repo for detailed descriptions and recommended usage per image.
